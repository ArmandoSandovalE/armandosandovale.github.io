Practicas

Practica 1: Presentación

La primera Practica, consistio en una introduccion a HTML, vimos la sintaxis basica de HTML, aprendimos 3 principales etiquetas: h1, p, small.

Practica 2: Hipervínculos en HTML

Aprendimos a darle un diseño a nuestra pagina WEB, aprendimos a introduccir Hipervínculos, un parrafo con una imagen, y por ultimo aprendimos lo que es un footer.

Practica 3: Formulario en HTML

Vimos los diferentes inputs que existen en HTML, que nos ayudan a diseñar formularios para nuestra pagina WEB, algunos ejemplos de ellos son: text, number, date, email, password, radio, checkbox, etc.

Practica 4: Bootstrap

Nos introducciomos en Bootstrap, que es una herramienta framework front-end utilizado para desarrollar aplicaciones web y sitios mobile first, o sea, con un layout que se adapta a la pantalla del dispositivo utilizado por el usuario.

Practica 5: Bootstrap - NavBar

La ultima practica consistio en aprender a desarrollar un navbar, que nos ayuda a tener un menu de navegación en nuestra pagina WEB, además colocamos un jumbotron, que es un cuadro grande para llamar la atención adicional sobre algún contenido o información especial.
